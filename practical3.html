Lists

We can create ordered and unordered lists in HTML using the `<ol>` and `<ul>` tags, respectively.

Ordered List (Top 3 Fruits)

  1. Apple
  2. Mango
  3. Banana

Unordered List (Shopping Items)

  • Milk
  • Eggs
  • Bread

Hyperlinks

Links are created using the anchor `<a>` tag with the `href` attribute.

  • External Link: Visit [Google](https://www.google.com) to perform a search.
  • Internal Link: Jump to the [Top of the page](#top). (Note: I added an id="top" to the body tag in the final code)
  • Email Link: Send an [email to example@example.com](mailto:example@example.com).

Images

Images are added using the `<img>` tag, which is an empty element.

The `src` attribute specifies the image source, and the `alt` attribute provides alternative text for accessibility.
